# Mangalam HDPE Pipes Landing Page

A responsive static product landing page for Mangalam HDPE Pipes. The page is built with plain HTML, CSS, and JavaScript, and includes product-focused sections, interactive carousels, modal forms, FAQ accordion, and mobile navigation.

## Features

- Responsive product landing page layout
- Sticky product call-to-action bar on scroll
- Hero image carousel with thumbnail navigation
- Product/application carousel with touch swipe support
- Image zoom preview on hero product image hover
- Process tabs and FAQ accordion
- Quote and catalogue request modals
- Smooth anchor scrolling with header offset

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

No build tools or package installation are required.

## How To Run Locally

Open `index.html` directly in your browser.

You can also use a lightweight local server from the project folder:

```powershell
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Deployment

This is a static website, so it can be deployed on GitHub Pages, Netlify, Vercel, or any static hosting service.

For GitHub Pages:

1. Push the code to GitHub.
2. Open the repository settings.
3. Go to **Pages**.
4. Select the `main` branch and root folder.
5. Save the settings.
